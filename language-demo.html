Select Language:

Choose your preferred language for AI interactions. The system will automatically translate responses and adapt voice synthesis accordingly.

Multi-Language Features

1. Language Detection

Automatically detects the language of user input (English/Marathi)

Example:
Input: "I am feeling stressed" → Detected: English
2. AI Response Translation

AI responses are automatically translated to user's preferred language

English Response:
"I understand you're feeling stressed. Try some deep breathing exercises."
3. Language-Aware Voice Synthesis

Text-to-speech adapts pronunciation and speed for different languages

Voice Parameters:
English: Normal speed (1.0x) | Marathi: Slower speed (0.9x) for clarity
4. Multilingual Speech Recognition

Speech-to-text supports both English and Marathi with automatic language detection

Supported: Whisper model with multilingual capabilities
5. Localized Crisis Resources

Crisis detection and resources are provided in the user's language

Crisis Resources:
National Suicide Prevention Lifeline: 988

Technical Implementation

  • Language Service: Manages language detection, translation, and model selection
  • Enhanced AI Services: Updated to support language parameters
  • Frontend Integration: Language selector and automatic language switching
  • API Enhancement: All endpoints now accept language parameters
